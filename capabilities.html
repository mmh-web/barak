Capabilities & Integrations

Everything Barak can do today — built-in tools, connected services, and what's available to connect next.

01

Core Tools

Built-in capabilities that are always available. These run directly on Barak's server — no external APIs needed.

Linux Shell

Active

Full terminal access — git, python3, node, curl, jq. Run any command.

File System

Active

Persistent workspace at /workspace. Read, write, edit files. Everything survives across sessions.

Web Search

Active

Search the internet via Brave API. Returns titles, URLs, and snippets for fast research.

Web Fetch

Active

Fetch and extract readable content from any URL. HTML to markdown conversion.

Headless Browser

Active

Full Chromium browser — navigate, screenshot, scrape, and interact with any web page.

Cron Scheduler

Active

Schedule recurring tasks — daily briefings, periodic syncs, automated monitoring.

Memory System

Active

Semantic search across markdown memory files. Two layers: conversation context + persistent files on disk.

Lossless Context

Active

Advanced conversation compression that preserves details in a searchable DAG. Never lose important context.

Sub-agents

Active

Spawn parallel workers for coding, research, or heavy tasks. Divide and conquer.

Text-to-Speech

Active

Convert text to audio. Generate voice messages and audio content on demand.

02

Communication Channels

Telegram Active
Primary channel. Send and receive messages, reactions, polls, inline buttons. This is where most interaction happens.
Slack Configured
Connected but not primary. Can send messages, react to posts, pin items, and search channels.
