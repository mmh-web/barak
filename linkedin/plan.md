# LinkedIn Engagement Plan — Michael Matias / Clarity

## Created: 2026-04-08
## Principals: Menachem (content writer/strategist), Michael Matias (CEO, public face)

## Strategic Positioning
Clarity's IP = the convergence of background + identity + behavior + deepfake detection = **human risk**
Goal: Expand brand from hiring fraud protection → employee identity verification across the full lifecycle

## Four Buyer Lanes
1. **Cybersecurity** — Checkpoint, Palo Alto, CrowdStrike. Gap: secure networks/endpoints but not the human.
2. **Identity Verification** — Clear, Persona, Onfido, Authentix. Gap: point solution, no risk profile.
3. **Background Check** — Checkr, First Advantage, HireRight. Gap: backward-looking, static, easy to bypass.
4. **HR/Remote Workforce** — Remote.com, Rippling. Gap: enable global hiring but can't verify identity.

## Core Narrative
Each lane solves one piece. Nobody stitches together the full human risk picture. Clarity has.
